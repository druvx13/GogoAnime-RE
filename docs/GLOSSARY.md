# Glossary

**OFFICIAL DOCUMENTATION**

## A
-   **Admin Panel:** The restricted area of the website used for content and user management, located at `/admin`.
-   **Anime:** Japanese animated productions. The core content entity of this system.

## C
-   **Controller:** In this system, typically a PHP file at the root level (e.g., `anime-details.php`) that handles the request logic.
-   **CRUD:** Create, Read, Update, Delete. The four basic functions of persistent storage.

## E
-   **Embed Code:** HTML code (usually an `<iframe>`) provided by a video host to display video on another site.
-   **Episode:** A single installment of an anime series.

## G
-   **Genre:** A category used to classify anime (e.g., Action, Romance).

## O
-   **Ongoing:** A status indicating an anime series is currently airing new episodes.

## P
-   **PDO:** PHP Data Objects. A database access layer providing a uniform method of access to multiple databases.
-   **Provider:** An external service that hosts video files (e.g., StreamSB, Doodstream).

## S
-   **Slug:** The part of a URL which identifies a particular page on a website in a form readable by users (e.g., `naruto-shippuden`).
-   **Stream:** The continuous transmission of audio or video files from a server to a client.

## V
-   **View:** The visual representation of data (HTML/CSS).
